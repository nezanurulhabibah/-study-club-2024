Perancangan dan Implementasi Sistem Pendukung Keputusan Pemilihan Smartphone Menggunakan Metode Simple Multi Attribute Rating Technique Berbasis Web

Pemilihan smartphone yang sesuai dengan kebutuhan dan preferensi pengguna seringkali menjadi tantangan karena banyaknya pilihan yang tersedia di pasar. Setiap smartphone memiliki spesifikasi dan fitur yang berbeda, seperti performa, kapasitas baterai, kamera, dan harga. Oleh karena itu, diperlukan sebuah sistem pendukung keputusan (SPK) yang dapat membantu pengguna memilih smartphone terbaik berdasarkan kriteria-kriteria yang relevan. Salah satu metode yang dapat digunakan untuk membangun SPK adalah Simple Multi Attribute Rating Technique (SMART). Metode SMART adalah salah satu teknik pengambilan keputusan yang sederhana namun efektif, di mana setiap alternatif dinilai berdasarkan atribut-atribut yang memiliki bobot tertentu. Pada artikel ini, akan dibahas bagaimana perancangan dan implementasi sistem pendukung keputusan pemilihan smartphone berbasis web menggunakan metode SMART.

Metode Simple Multi Attribute Rating Technique (SMART)

SMART merupakan metode pengambilan keputusan dengan menggunakan pendekatan multi-atribut, di mana setiap alternatif (dalam hal ini smartphone) dinilai berdasarkan beberapa kriteria atau atribut. Metode ini melibatkan langkah-langkah sebagai berikut:

  • Penentuan Kriteria: Menentukan kriteria-kriteria yang digunakan untuk menilai smartphone. Kriteria ini dapat mencakup harga, kapasitas baterai, kualitas kamera, performa prosesor, ukuran layar, dan lainnya. Pembobotan Kriteria: Setiap kriteria diberi bobot yang menunjukkan tingkat kepentingannya. Misalnya, jika performa prosesor lebih penting daripada harga, bobot untuk kriteria prosesor akan lebih besar. Penilaian Alternatif: Setiap smartphone dinilai berdasarkan kriteria-kriteria tersebut. Nilai diberikan untuk setiap atribut berdasarkan performa smartphone dalam atribut tersebut. Perhitungan Nilai Total: Nilai total dari setiap alternatif dihitung dengan mengalikan nilai atribut dengan bobot masing-masing, kemudian menjumlahkan hasilnya. Pemilihan Alternatif Terbaik: Smartphone dengan nilai total tertinggi dipilih sebagai alternatif terbaik.
    • Sistem pendukung keputusan pemilihan smartphone ini dirancang berbasis web agar dapat diakses dengan mudah oleh pengguna. Sistem terdiri dari beberapa komponen utama: User Interface (UI): Tampilan antarmuka web yang intuitif dan mudah digunakan, memungkinkan pengguna memasukkan preferensi kriteria dan melakukan pemilihan smartphone. Antarmuka mencakup form input untuk bobot kriteria, serta tampilan hasil rekomendasi smartphone. Database: Database menyimpan data smartphone, termasuk spesifikasi, harga, dan fitur lainnya. Data ini digunakan untuk perhitungan dalam metode SMART. Algoritma SMART: Algoritma ini mengimplementasikan langkah-langkah metode SMART untuk menghitung nilai total dari setiap smartphone berdasarkan input kriteria dan bobot dari pengguna. Backend: Sistem backend menangani proses perhitungan dan manajemen data, serta menyediakan API untuk komunikasi antara UI dan database.

    Implementasi sistem dilakukan menggunakan bahasa pemrograman PHP dengan framework Laravel sebagai backend dan MySQL sebagai database. Berikut adalah tahapan implementasi sistem: Pengumpulan Data Smartphone: Data spesifikasi smartphone dikumpulkan dari berbagai sumber, seperti situs e-commerce dan portal teknologi, kemudian disimpan ke dalam database. Pengembangan Antarmuka Pengguna: UI dikembangkan menggunakan HTML, CSS, dan JavaScript dengan framework Bootstrap untuk tampilan yang responsif dan user-friendly. Pengguna dapat menginputkan bobot kriteria sesuai preferensi mereka melalui antarmuka ini. Perhitungan dan Rekomendasi: Sistem menerima input bobot kriteria dari pengguna dan menggunakan algoritma SMART untuk menghitung nilai total dari setiap smartphone. Hasil perhitungan ditampilkan dalam bentuk rekomendasi, di mana smartphone dengan nilai tertinggi ditempatkan di urutan pertama. Pengujian Sistem: Setelah implementasi, sistem diuji untuk memastikan bahwa hasil rekomendasi sesuai dengan preferensi pengguna. Pengujian dilakukan dengan berbagai skenario bobot kriteria yang berbeda.

    Setelah sistem diimplementasikan, hasilnya menunjukkan bahwa metode SMART dapat memberikan rekomendasi smartphone yang sesuai dengan kebutuhan pengguna. Pengguna dapat dengan mudah menyesuaikan bobot kriteria berdasarkan preferensi pribadi, seperti memprioritaskan harga, performa, atau fitur tertentu. Penggunaan web sebagai platform juga memudahkan akses dan penggunaan sistem oleh berbagai kalangan. Sistem ini dapat diakses melalui perangkat apapun yang mendukung browser, sehingga lebih fleksibel untuk digunakan.

    Sistem pendukung keputusan pemilihan smartphone berbasis web dengan menggunakan metode SMART berhasil dirancang dan diimplementasikan. Sistem ini dapat membantu pengguna dalam memilih smartphone yang paling sesuai dengan kebutuhan dan preferensi mereka. Dengan menggunakan pendekatan multi-kriteria, pengguna dapat mengevaluasi berbagai pilihan smartphone dengan lebih objektif dan efisien. Ke depan, sistem ini dapat dikembangkan lebih lanjut dengan penambahan fitur seperti integrasi data real-time dari situs e-commerce atau penambahan kriteria tambahan yang lebih spesifik. Hal ini akan meningkatkan akurasi rekomendasi serta kenyamanan pengguna dalam mencari smartphone yang diinginkan.
